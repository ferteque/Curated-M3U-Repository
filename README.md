EACH EPG HAS AN ID IN ITS NAME, THE ID CAN BE FOUND IN THE REPOSITORY: https://repository-server.onrender.com/


The main objective of this repository is that people can share their curated lists for different services, countries, and categories. Obviously, providing the EPG is a must to consider it curated. This way, we can ask premium services for more than curated lists and proper EPG.

YOU WILL NEED TO HAVE YOUR OWN SET OF CREDENTIALS FROM Z2U, ALIBABA OR ANY OTHER MARKETPLACE FOR THE LIST SERVICE YOU INTEND TO USE.

It is not intended only for Strong. It is intended for ANY service. The idea is that this can be something for the whole community to use, avoiding then overpriced services with the curated list or proper EPG excuse. Also, give the rest the opportunity to start customising their own lists from a better point.

For the moment, I just created a simple google sheet with the information and a Google Form for the upload requests. I want to thank all who tested with me for the help provided. Also want to thank the Strong 8k EPG guy from yesterday.

Now, how does this work:

UPLOAD REQUEST:

1. Download your M3U file from your IPTV Editor.

2. Open it with the windows notepad (or any other text editor)

3. Using the "Search and replace" feature, search by your credentials (i.e. http://xxxxx.xxxx-xxx/12345678/12345678) and replace it with http://DNS/USERNAME/PASSWORD

4. Make sure you are also doing it for Movies and TV Shows if you are including them in your lists (i.e. http://xxxx.xxx-xxx/movie/12345678/12345678) and replace with http://DNS/movie/USERNAME/PASSWORD

5. Make an upload request to this Google Form: https://forms.gle/ufrnmrwF5RhGZXyJA

USING AN M3U FILE

1. Download the M3U file that fits you best: https://repository-server.onrender.com/

2. Open it with the Windows notepad (or any other text editor)

3. Using the "Search and replace" feature, search by http://DNS/USERNAME/PASSWORD and replace it with your credentials (i.e. http://xxxxx.xxxx-xxx/12345678/12345678)

4. Make sure you are also doing it for Movies and TV Shows if they are included in that list searching by http://DNS/movie/USERNAME/PASSWORD and replacing it with your credentials (i.e. http://xxxx.xxx-xxx/movie/12345678/12345678)

5. Upload the M3U file to your IPTV player. You can also use it in your IPTV Editor as a starting point.

6. If you are going to add the EPG to IPTV Editor, use the GitHub link, it is updated once a day (3 a.m. UTC). It may fail when adding it, retry until it works. If it says the EPG is empty, delete it and add it again, until it works.

DISCLAIMERS:

- If you have a reduced service, for example, Strong 8k without UK content from the reseller, you will not be able to access UK channels, even if the M3U file includes them.

- I am not responsible for any issues that these lists may have. I will simply be making sure that files shared are really M3U lists.
